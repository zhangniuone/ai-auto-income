## ⚡ 快速开始

### 1. 部署到服务器

```bash
# 1. 克隆项目
git clone https://github.com/zhangniuone/ai-auto-income.git
cd ai-auto-income

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 填入你的 API Keys 和广告代码

# 3. Docker 一键启动
docker-compose up -d

# 4. 访问你的网站
# http://your-domain.com
```

### 2. 配置广告（收入来源）

```env
# Google Adsense
GOOGLE_ADSENSE_CLIENT=ca-pub-xxxxxxxxxxxx
GOOGLE_ADSENSE_SLOT=xxxxxxxxx

# 百度联盟
BAIDU_ADS_SLOT=xxxxxx

# 淘宝客/京东联盟
TAOBAO_PID=mm_xxxxx_xxxxx_xxxx
JD_UNION_ID=xxxxxx
```

### 3. 选择内容领域

编辑 `config/niche.json`：

```json
{
  "niche": "tech",
  "language": "zh",
  "topics": [
    "人工智能",
    "ChatGPT",
    "AI工具推荐",
    "效率软件",
    "编程技巧"
  ],
  "keywords": [
    "AI教程",
    "ChatGPT使用技巧",
    "AI赚钱方法"
  ]
}
```

## 📊 系统功能

### ✅ 自动内容生成
- **热点监测** - 自动抓取百度热搜、微博热搜、知乎热榜
- **AI 写作** - 调用 Kimi/GPT-4 生成高质量原创文章
- **自动配图** - AI 生成或搜索配图
- **多语言支持** - 中文/英文/日文自动切换

### ✅ SEO 全自动优化
- **自动关键词挖掘** - 长尾关键词分析
- **内链自动生成** - 相关文章自动链接
- **Sitemap 自动更新** - 提交搜索引擎

### ✅ 流量变现
- **广告位自动插入** - 文章中智能插入广告
- **联盟链接自动替换** - 产品名自动加推广链接
- **A/B 测试** - 自动优化广告位置提高点击率

### ✅ 数据分析
- **访问统计** - 实时流量监控
- **收入统计** - 各渠道收入汇总
- **文章表现** - 自动分析热门内容方向
- **自动优化** - 根据数据调整内容策略

```json
{
  "site": {
    "name": "AI工具导航",
    "description": "发现最好用的AI工具",
    "domain": "https://your-domain.com"
  },
  "content": {
    "articlesPerDay": 10,

    "adsEnabled": true,
    "affiliateEnabled": true,
    "adDensity": "medium"
  }

```

## 🎯 运行原理

1. **定时触发** (每4小时)
2. **热点抓取** - 从各平台获取热门话题
3. **关键词分析** - 选择有搜索量的话题
4. **AI 写作** - 生成1500-3000字原创文章
5. **SEO 优化** - 自动插入关键词、内链
6. **发布上线** - 自动发布到网站
7. **提交收录** - 自动通知搜索引擎
8. **广告展示** - 访客阅读，产生广告收入

**全程自动化，无需人工干预！**
